# rails5-chatroom

## Description
ICCS474 Project 1 in T2/2015-2016

## Installation
Make sure you already installed redis in your computer. 

For example, run the following command if you use Mac OSX 

```bash
$ brew install redis
```

## Version
* ruby: 2.2.2
* rails: 5.0.0.beta1

## Starting server
```bash
$ foreman start
```
